📚 Legenda - Kako funkcioniše linkovanje
💰
Kalkulacija - Osnova svega. Sadrži sve parametre, cene, materijale.
📋
Glavni nalog - Povezan sa kalkulacijom (kalkulacija_id). Sadrži sve ukupne podatke.
🔗
Pojedinačni nalozi - Svaki ima 2 linka: glavni_nalog_id + kalkulacija_id
✨ Prednosti ovog sistema
  • Dupla veza: Možeš pristupiti nalozima direktno iz kalkulacije ILI iz glavnog naloga
  • SQL Query 1: SELECT * FROM stampa_nalozi WHERE kalkulacija_id = '...' → Sve štampe za kalkulaciju
  • SQL Query 2: SELECT * FROM stampa_nalozi WHERE glavni_nalog_id = '...' → Štampa za nalog
  • Kompletan pregled: VIEW kompletan_nalog_view spaja SVE tabele automatski
  • Brisanje: Ako obrišeš glavni nalog → CASCADE briše sve pojedinačne naloge
  • Fleksibilnost: Možeš dodati nove tipove naloga bez promene strukture
  • Praćenje: Svaki nalog ima svoj status (čeka, u radu, završen)
  • Istorija: created_at, updated_at na svakom nalogu
